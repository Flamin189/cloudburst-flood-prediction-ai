🌊 Flash Flood Prediction

Upload environmental data to predict flood risk
📋 CSV File Requirements

Your CSV file must include the following columns:

  • rainfall - Rainfall amount (0-100)
  • humidity - Humidity percentage (0-100)
  • temperature - Temperature in Celsius (-50 to 60)

The system will use the first row of data for prediction.

📝 Example CSV Format
rainfall,humidity,temperature
75.5,85.2,22
45.0,60.1,18
90.0,95.5,15
Only .csv files are accepted
✨ Features
  • Machine Learning Prediction: Uses RandomForestClassifier trained on environmental patterns
  • Multi-level Risk Assessment: Low, Medium, or High flood risk
  • Confidence Scoring: See how confident the model is in its prediction
  • Integrated Alerts: Combined analysis with cloudburst detection for comprehensive hazard assessment
  • Email Notifications: Automatic alerts sent for critical conditions

MODEL TRAINING DETAILS
──────────────────────────────────────────────────────────────────────────────

Training Script: train_flood_model.py

Input Requirements:
├── CSV File Location: data/flood_training_data.csv
├── Required Columns: rainfall, humidity, temperature, flood_risk
├── Data Type: Numeric (float) for features, Text for target
└── Sample Size: Minimum 100 records recommended

Process:
1. Load CSV and validate columns
2. Encode flood_risk labels (Low→0, Medium→1, High→2)
3. Split data: 80% train, 20% test
4. Train RandomForestClassifier
5. Evaluate performance (accuracy, precision, recall)
6. Save model and encoder to pickle files

Output Files:
├── models/flood_model.pkl (~500KB-2MB)
└── models/flood_label_encoder.pkl (~1KB)
